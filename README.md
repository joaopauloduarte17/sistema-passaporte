# Sistema Passaporte

Sistema web desenvolvido como atividade acadêmica utilizando Java Spring Boot no back-end e HTML, CSS, Materialize e jQuery no front-end.

## Funcionalidades

- Cadastro de usuários
- Login de usuário
- Validação de usuário do tipo RH
- Cadastro de Perfil Passaporte
- Listagem dos Perfis Passaporte cadastrados pelo usuário logado
- Edição de Perfil Passaporte
- Integração entre front-end e back-end via requisições AJAX
- Persistência de dados com Spring Data JPA e banco H2

## Tecnologias utilizadas

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- HTML
- CSS
- Materialize
- jQuery
- JavaScript

## Como executar

1. Clone o repositório: git clone link-do-repositorio.
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe principal: Application.java.
